# Applications of Machine Learning

Machine Learning is not a single product.
It is a **problem-solving approach** used across industries wherever:

* Data exists
* Patterns exist
* Decisions need automation
* Scale is large

Now we’ll go domain by domain.

* Health risk scoring

### Type of ML Used:

* Supervised learning (classification)
* Deep learning (CNN for images)
* Reinforcement learning (drug molecule optimization)

### Industry Thinking:

* Data is sensitive (HIPAA issues)
* Labeling medical images is expensive
* False negatives can cost lives

If your model has 95% accuracy but misses cancer cases, it's useless.

# Now Let Me Challenge You

Don’t just memorize applications.

For each domain, you must think:

1. What is the prediction target?
2. What data is required?
3. What is the failure cost?
4. Is it batch or real-time?
5. How does it scale?
6. How do we monitor drift?

If you cannot answer these, you don’t understand applications.
